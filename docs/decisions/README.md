# Architecture Decision Records (ADRs)

This folder contains Architecture Decision Records (ADRs) for AccessGuard.

ADRs capture significant architectural decisions, along with context, consequences, and alternatives.
They are intended to be short, durable, and updated only when decisions change.

Naming convention:
- `ADR-01-<short-title>.md`

ADR status values:
- Proposed
- Accepted
- Superseded
